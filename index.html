MÓDULOS PEDAGÓGICOS (6 Sesiones)

Módulo 1: Sensores Arduino y Adquisición KDD: Adquisición
Objetivo de Aprendizaje:

Implementar sistema Arduino para lectura confiable de sensores DHT22 y almacenamiento local.

Contenidos (4 horas):

  • Arquitectura básica Arduino para adquisición
  • Sensor DHT22: conexión y calibración
  • Lectura serial y almacenamiento en SD
  • Protocolo básico de comunicación

Actividad Práctica:

Crear sistema que registre temperatura y humedad cada 5 minutos durante 24 horas.

Entregable Pedagógico: Dataset CSV con 288 registros válidos
Módulo 2: Python como Puente ETL KDD: Preprocesamiento
Objetivo de Aprendizaje:

Desarrollar script Python que lea datos seriales, los limpie y cargue a PostgreSQL.

Contenidos (4 horas):

  • PySerial para comunicación Arduino-Python
  • Limpieza básica de datos: outliers y valores nulos
  • Conexión Python-PostgreSQL con psycopg2
  • Inserción eficiente de datos

Actividad Práctica:

Crear pipeline ETL que reciba datos en tiempo real y los almacene en PostgreSQL.

Entregable Pedagógico: Script Python funcional + BD con datos en tiempo real
Módulo 3: PostgreSQL para Series Temporales KDD: Almacenamiento
Objetivo de Aprendizaje:

Diseñar esquema PostgreSQL optimizado para series temporales IoT.

Contenidos (4 horas):

  • Diseño de tablas para datos sensoriales
  • Índices para consultas temporales
  • Consultas SQL para análisis básico
  • Vistas materializadas para KPIs

Actividad Práctica:

Crear BD normalizada con tablas: sensores, lecturas, ubicaciones.

Entregable Pedagógico: Esquema SQL + 3 consultas analíticas básicas
Módulo 4: Power BI para Visualización KDD: Interpretación
Objetivo de Aprendizaje:

Crear dashboard Power BI conectado a PostgreSQL con KPIs relevantes.

Contenidos (4 horas):

  • Conexión Power BI - PostgreSQL
  • DAX para cálculos temporales
  • Visualizaciones para series temporales
  • Alertas y umbrales visuales

Actividad Práctica:

Dashboard con: gráfico temporal, KPI actual, alertas, histórico 24h.

Entregable Pedagógico: Dashboard Power BI funcional y documentado
Módulo 5: KDD Aplicada - Análisis KDD: Minería
Objetivo de Aprendizaje:

Aplicar análisis KDD para detectar patrones y anomalías en los datos.

Contenidos (4 horas):

  • Análisis exploratorio en Python
  • Detección de outliers estadísticos
  • Patrones temporales básicos
  • Integración resultados en Power BI

Actividad Práctica:

Script Python que analice datos históricos y genere reporte KDD.

Entregable Pedagógico: Reporte KDD con 3 insights identificados
Módulo 6: Integración Final KDD: Aplicación
Objetivo de Aprendizaje:

Integrar sistema completo Arduino → Python → PostgreSQL → Power BI.

Contenidos (4 horas):

  • Pruebas de integración
  • Optimización del pipeline
  • Documentación técnica
  • Presentación de resultados

Actividad Práctica:

Demostración sistema funcionando con datos en tiempo real.

Entregable Pedagógico: Sistema completo + Video demo 3 minutos
